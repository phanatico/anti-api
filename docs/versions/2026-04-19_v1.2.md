# ANTI-API Panel v1.2

## Cambios principales

- Añadido login/logout y estado de sesión para cuentas.
- Añadida validación de cuenta y soporte de verificación.
- Añadida gestión de cookies por modelo con almacenamiento JSON específico por usuario.
- Añadido endpoint `POST /api/model/cookies` para guardar cookies y `GET /api/model/cookies` para cargarlas.
- Integrado panel web con campos de cookies y acciones de guardar/cargar.
- Actualizada lógica de `AntiApiChat` para usar rutas de cookies específicas por cuenta y modelo.
- Versionado de debug logs en `docs/debug` con `v1.2`.

- `app.py`
- `static/app.js`
- `templates/index.html`
- `models.json`
