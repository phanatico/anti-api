# ANTI-API - Versión v1.1

Fecha: 2026-04-19
Hora: 00:00

## Cambios incluidos

- Añadido sistema de debug automático.
  - Se guarda un archivo de `docs/debug/<fecha>/session-<hora>_<id>.jsonl` por sesión.
  - Cada entrada registra evento, modelo, prompt, respuesta, error y metadatos de sesión.
- Añadido sistema de cuentas con JSON.
  - `accounts.json` se usa para almacenar usuarios.
  - Se implementaron endpoints de registro y verificación:
    - `/api/account/register`
    - `/api/account/verify`
- Añadido panel de cuenta en la interfaz web.
  - Registro de email y contraseña.
  - Código de verificación devuelto al crear cuenta.
  - Verificación desde la UI.
- Se mantuvo el servidor en el puerto `4000`.

## Archivos nuevos

- `accounts.json`
- `docs/versions/2026-04-19_v1.1.md`
- `docs/debug/` (carpeta de logs automáticos)

## Notas

- El registro de cuentas se guarda en `accounts.json`.
- Los eventos de debug se añaden por sesión y fecha en archivos JSONL dentro de `docs/debug`.
- Este documento es la primera versión de cambios importantes, por eso se etiqueta como `v1.1`.
