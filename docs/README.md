# Documentación del proyecto ANTI-API

## Estructura

- `docs/versions/` - historial de versiones con fecha y características.
- `docs/debug/` - logs automáticos de sesión generados en cada ejecución.

## Uso

- Arrancar el panel web local con `run_panel_debug.bat`.
- La app ejecuta en `http://127.0.0.1:4000`.
- Los archivos de debug se guardan automáticamente en `docs/debug/<fecha>/`.

## Versiones

- `v1.1` - cuenta de usuario, verificación y logging por sesión.
